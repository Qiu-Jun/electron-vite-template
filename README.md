# electron-vite-template
基于vite的electron脚手架

## 工具库说明
+ kill-port: 清理端口
+ cross-env: 设置环境变量
+ npm-run-all: 顺序执行script脚本
+ concurrently: 并行执行script脚本
+ tsc-watch: 编译ts文件,并在文件修改后重新执行编译
+ wait-on: 等待文件/端口等变化后执行script脚本
